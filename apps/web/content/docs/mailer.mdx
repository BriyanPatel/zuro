Send emails with a single `sendMail()` function. Supports SMTP (Nodemailer) and Resend.

---

## Installation

<Tabs items={['npm', 'pnpm', 'bun']}>

  </Tab>
  <Tab value="pnpm">

bun x zuro-cli add mailer

## Usage

**Send an email:**

```ts
import { sendMail } from "../lib/mailer";

await sendMail({
  to: "user@example.com",
  subject: "Welcome!",
  html: "<h1>Welcome to our app</h1>",
});
```

**Send with a template:**

```ts
await sendMail({
  to: "user@example.com",
  subject: "Welcome!",
  template: "welcome",
  data: { name: "Briyan" },
});
```

**Send to multiple recipients:**

```ts
await sendMail({
  to: ["alice@example.com", "bob@example.com"],
  subject: "Announcement",
  html: "<p>Big news!</p>",
});
```

**Create a custom template:**

```ts title="src/lib/mail-templates/reset-password.ts"
export const resetPassword = (data: Record<string, unknown>) => {
  const link = typeof data.link === "string" ? data.link : "#";
  return `
    <h1>Reset Your Password</h1>
    <a href="${link}">Reset Password</a>
  `;
};
```

Register it in `src/lib/mail-templates/index.ts`:

```ts
import { welcome } from "./welcome";
import { resetPassword } from "./reset-password";

export const templates = { welcome, "reset-password": resetPassword };
```

## Configuration

### SMTP

```bash
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=your-email@gmail.com
SMTP_PASS=your-app-password
MAIL_FROM=noreply@yourapp.com
```

### Resend

```bash
RESEND_API_KEY=re_your_api_key
MAIL_FROM=onboarding@resend.dev
```

<Callout type="info">
  For Gmail, enable "App Passwords" in your Google Account settings. Regular
  passwords won't work.
</Callout>

## Troubleshooting

- **`Email template 'x' not found`** — make sure it's exported in `src/lib/mail-templates/index.ts`
- **SMTP connection refused** — check `SMTP_HOST` and `SMTP_PORT` in `.env`
- **Placeholder values** — update dummy values in `.env` before sending real emails
