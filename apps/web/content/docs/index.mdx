# What is Zuro?

**Zuro** is a modern CLI toolkit that scaffolds production-ready Node.js backends with a single command. It bridges the gap between starting from scratch and using heavy frameworks, giving you the perfect balance of control and productivity.

Think of it as **Create React App** for backend development - but better. You get type-safe, secure, and modular code that you fully own and understand.

## Quick Start

Get started with Zuro in seconds:

```bash
npx zuro-cli init
```

This will:
1. Set up a new project with TypeScript
2. Install Express.js and essential dependencies
3. Generate a production-ready server structure
4. Configure security, validation, and environment handling
