# validator

## Overview

Add production-ready request validation to your backend with a single middleware.

---

## Install

```bash
npx zuro add validator
```

Adds `validate()` middleware so routes can validate `body`, `query`, and `params` with Zod.

---

## What This Generates

```text
src/
└ middleware/
  └ validate.ts
```

`src/middleware/validate.ts` exports `validate(schema)`, parses incoming request payloads, rewrites `req.body/query/params` with parsed values, and returns `422` for invalid input.

---

## Quick Example

```ts
import { Router } from "express";
import { z } from "zod";
import { validate } from "../middleware/validate";

const router = Router();

const createUserSchema = z.object({
  body: z.object({
    name: z.string().min(1),
    email: z.string().email(),
  }),
});

router.post("/users", validate(createUserSchema), async (req, res) => {
  res.status(201).json({
    success: true,
    user: req.body,
  });
});
```

Example request:

```http
POST /users
Content-Type: application/json

{
  "name": "Ava",
  "email": "not-an-email"
}
```

Example response:

```json
{
  "status": "error",
  "code": "VALIDATION_ERROR",
  "message": "Validation failed",
  "errors": [
    {
      "path": "body.email",
      "message": "Invalid email"
    }
  ]
}
```

---

## How It Works

Client -> Route middleware `validate(schema)` -> Zod `parseAsync({ body, query, params })` -> valid: request continues with parsed values -> invalid: `422 VALIDATION_ERROR`

---

## Configuration

```bash
# No required environment variables
```

`validator` is fully code-configured through your Zod schemas.

---

## API Reference

`validate(schema)`: Express middleware factory for request validation.
`schema.body`: validates and normalizes `req.body`.
`schema.query`: validates and normalizes `req.query`.
`schema.params`: validates and normalizes `req.params`.
`422 VALIDATION_ERROR`: automatic error response with `{ path, message }[]` issues.

## Advanced Usage

Validate query + params with coercion:

```ts
const listSchema = z.object({
  params: z.object({
    projectId: z.string().uuid(),
  }),
  query: z.object({
    page: z.coerce.number().int().positive().default(1),
    limit: z.coerce.number().int().positive().max(100).default(20),
  }),
});

router.get("/projects/:projectId/issues", validate(listSchema), (req, res) => {
  res.json({
    projectId: req.params.projectId,
    page: req.query.page,
    limit: req.query.limit,
  });
});
```

Reuse shared schema blocks:

```ts
const pagination = z.object({
  page: z.coerce.number().int().positive().default(1),
  limit: z.coerce.number().int().positive().default(20),
});

const schema = z.object({
  query: pagination,
});
```
