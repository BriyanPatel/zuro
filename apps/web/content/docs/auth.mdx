## Usage

**Sign up a user:**

```bash
curl -X POST http://localhost:3000/api/auth/sign-up/email \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Briyan",
    "email": "briyan@example.com",
    "password": "password123"
  }'
```

**Log in:**

```bash
curl -X POST http://localhost:3000/api/auth/sign-in/email \
  -H "Content-Type: application/json" \
  -d '{
    "email": "briyan@example.com",
    "password": "password123"
  }'
```

**Get current user's profile:**

```bash
curl http://localhost:3000/api/users/me \
  -H "Cookie: <session_cookie_from_login>"
```

**Available endpoints:**

| Path                      | Method | What it does    |
| ------------------------- | ------ | --------------- |
| `/api/auth/sign-up/email` | POST   | Register        |
| `/api/auth/sign-in/email` | POST   | Login           |
| `/api/auth/sign-out`      | POST   | Logout          |
| `/api/auth/get-session`   | GET    | Current session |
| `/api/users/me`           | GET    | User profile    |

**Protect your own routes:**

```ts
import { auth } from "../lib/auth";
import { fromNodeHeaders } from "better-auth/node";
import { UnauthorizedError } from "../lib/errors";
import { asyncHandler } from "../middleware/error-handler";

router.get(
  "/dashboard",
  asyncHandler(async (req, res) => {
    const session = await auth.api.getSession({
      headers: fromNodeHeaders(req.headers),
    });

    if (!session) throw new UnauthorizedError("Not authenticated");

    res.json({ message: `Welcome, ${session.user.name}!` });
  }),
);
```

## Configuration

Two env vars are added automatically:

```bash
BETTER_AUTH_SECRET=<auto-generated-64-char-hex>
BETTER_AUTH_URL=http://localhost:3000
```

---

## Troubleshooting

- **Migration errors** — check `DATABASE_URL` is correct and the database is running
- **401 on `/api/users/me`** — include the session cookie from the login response
- **`BETTER_AUTH_URL` issues** — must match your backend origin exactly
