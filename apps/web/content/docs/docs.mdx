## Usage

After install, visit:

- **Scalar UI:** `http://localhost:3000/api/docs`
- **OpenAPI JSON:** `http://localhost:3000/api/docs/openapi.json`

**Document your own routes** in `src/lib/openapi.ts`:

```ts title="src/lib/openapi.ts"
import { z } from "zod";

const createPostSchema = z.object({
  title: z.string().min(1),
  content: z.string().min(1),
});

registry.registerPath({
  method: "post",
  path: "/api/posts",
  tags: ["Posts"],
  request: {
    body: {
      content: {
        "application/json": { schema: createPostSchema },
      },
    },
  },
  responses: {
    201: { description: "Post created" },
  },
});
```

Refresh `/api/docs` and your new endpoint appears with full schema details.

---

## Configuration

No environment variables needed. Auth endpoints are added to OpenAPI automatically when both `auth` and `docs` modules are installed.

---

## Troubleshooting

- **Docs page blank** — make sure `docs.routes.ts` is imported in your router
- **Endpoint not showing** — verify it's registered in `src/lib/openapi.ts` and restart the server
- **Auth endpoints missing** — run `add auth` after `add docs` (or vice versa) to sync them
