# API Docs

Add interactive API docs with Scalar and OpenAPI generation from Zod schemas.

---

- OpenAPI JSON: `http://localhost:3000/api/docs/openapi.json`

## How It Works

- `src/lib/openapi.ts` creates an OpenAPI document using Zod.
- `src/routes/docs.routes.ts` serves OpenAPI JSON and Scalar UI.
- Zuro injects docs routing into your API router automatically.

---

## Add Your Own Zod Docs

Inside `src/lib/openapi.ts`, register your route with Zod schemas:

```ts
const createPostSchema = z.object({
  title: z.string().min(1),
  content: z.string().min(1),
});

registry.registerPath({
  method: "post",
  path: "/api/posts",
  tags: ["Posts"],
  request: {
    body: {
      content: {
        "application/json": {
          schema: createPostSchema,
        },
      },
    },
  },
  responses: {
    201: { description: "Post created" },
  },
});
```

---

## Auth Integration

- If auth is already installed, `add docs` also adds auth endpoints to OpenAPI.
- If docs already exists, `add auth` updates OpenAPI docs automatically.
- If docs is missing, `add auth` asks whether to install docs too (default: yes).
