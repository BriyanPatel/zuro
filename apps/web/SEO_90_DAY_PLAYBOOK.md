# SEO 90-Day Execution Playbook

## Day 1-14: Technical foundation
- Set `SITE_URL`, `GOOGLE_SITE_VERIFICATION`, and `GA4_MEASUREMENT_ID` in production.
- Verify Search Console domain property and submit `/sitemap.xml`.
- Validate robots, canonicals, OG/Twitter metadata, and structured data.
- Run `corepack pnpm --filter web seo:audit` and fix blocking SEO assertions.

## Day 15-60: Content production
- Publish and improve commercial landing pages monthly.
- Maintain 4-8 net new pages/month aligned to primary and secondary keywords.
- Refresh top docs pages with clearer intros, examples, FAQs, and internal links.
- Track CTR and average position for landing/docs pages in Search Console.

## Day 30-90: Authority and distribution
- Publish and iterate comparison pages.
- Secure 20-30 quality backlinks from dev communities, ecosystem directories, and technical writeups.
- Keep canonical docs URLs in GitHub README, profile links, and release notes.

## Weekly operating cadence
- Review top queries and pages in Search Console.
- Improve titles/meta descriptions for low-CTR, high-impression pages.
- Add internal links from high-traffic pages to high-conversion pages.
- Update stale docs with clearer examples and intent-focused intros.

## Monthly operating cadence
- Identify keyword cannibalization and merge/split pages as needed.
- Review CWV and Lighthouse trends on homepage/docs/commercial pages.
- Prioritize next content batch from ranking opportunities and conversion data.
